rs-sc 设置 配置 OpenAI 兼容接口与全局快捷键

支持任意 OpenAI 兼容服务: OpenAI / DeepSeek / Ollama / Azure 等

格式: Ctrl/CommandOrControl + Shift + Alt + 字母/数字/F1-F12, 用 + 连接,例 Ctrl+Alt+Q。点"录制"后按想要的组合。

用于随时打开本设置面板。默认 Ctrl+Shift+P——托盘图标在 Win11 上有时会被自动折叠到通知区不可见, 所以再给你一条直接进设置的快捷键兜底。

触发"数位板手写提问"流程:截屏 → 框选 → 在弹出窗口里用笔写字 → LLM 识别后回答。 默认 Ctrl+Shift+N。必须和上面两个快捷键都不同。

第一步:把用户手写内容识别成纯文本

第二步:把识别出的文字 + 原始截图一起发给模型,让它回答
